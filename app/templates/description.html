Datasets

The monitoring system currently includes the following stations:

  • Kilpisjärvi
  • Lammi
  • Tvärminne

Each of the stations contains datasets with different recorded environmental parameters. For example, for Kilpisjärvi station it is possible to obtain the following datasets:

  • Dataset id "a0c62d62-c0b3-4c3e-b82f-fb2f1c99b941" - Bilberry monitoring at Kilpisjärvi sub-arctic station, Northern Finland
  • Dataset id "d79f09a0-f8d8-4328-8b21-38bb15d72357" - Mountain birch (Betula pubescens tortuosa) monitoring at Kilpisjärvi sub-arctic station, Northern Finland

Licence

Unless otherwise specified, datasets are distributed under the "Attribution-NonCommercial 4.0 International" license. However, please note that some datasets are distributed under licenses specific only to them. This will be explicitly stated in the metadata.

Data provenance

Each dataset that is presented in this catalog has an owner. These owners are Finnish research institutes that provide a detailed description of the data provenance. This information is available in the description of each dataset and in its metadata. If you have any questions, please contact the contacts listed in the metadata.
